# How to Update Your GitHub Profile

Follow these steps to make your profile look awesome!

## Step 1: Create a New Repository
1.  Go to [GitHub](https://github.com) and log in.
2.  Click on the **+** icon in the top-right corner and select **New repository**.
3.  **Repository Name**: Enter `Atul-Chahar`.
    *   *Note: GitHub will show you a secret message saying "You found a secret! Atul-Chahar/Atul-Chahar is a special repository..."*
4.  **Public**: Make sure the repository is **Public**.
5.  **Initialize**: Check the box **"Add a README file"**.
6.  Click **Create repository**.

## Step 2: Update the Content
1.  Navigate to your newly created repository (`https://github.com/Atul-Chahar/Atul-Chahar`).
2.  Click the **Pencil icon** (Edit) to edit the `README.md` file.
3.  Delete all existing content in that file.
4.  Copy the entire content from the `README.md` file provided here.
5.  Paste it into the GitHub editor.

## Step 3: Customize (Optional)
-   **Icons**: I included icons for JS, Python, React, Node, HTML, CSS, Git, and Linux. If you want to change these, you can search for other icons on [Devicon](https://devicon.dev/) or remove the lines you don't need.
-   **Social Links**: I added your email. If you have a LinkedIn or Twitter profile, uncomment those lines (remove `<!--` and `-->`) and replace the URLs with your own.

## Step 4: Save
1.  Scroll down to the "Commit changes" section.
2.  Click the green **Commit changes** button.

🎉 **Done!** Go to your profile page (`https://github.com/Atul-Chahar`) to see your new awesome look.
